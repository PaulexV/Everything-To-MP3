# Description

Everything-To-MP3 is a project that aims to offer a service that transforms, either a link, or anything else to a playable mp3.

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun start

# watch mode
$ bun start --watch

# production mode
$ bun start:prod
```

## Test

```bash
# unit tests
$ bun test

# e2e tests
$ bun test:e2e

# test coverage
$ bun test:cov
```
